#### Caldo de Huesos {#caldo-de-huesos}

**Condición de Activación:**
```
EXCEDENTE:huesos (>5kg/semana) + CLIENTES:8+
```

**Origen:** Huesos de vaca/pollo de los pedidos de clientes.

**Producto:**
- Caldo concentrado ("Liquid Gold")
- Gelatina de colágeno
- Demi-glace

**Proceso:**
1. Juntar huesos de 2 semanas
2. Tostar en horno (30 min)
3. Hervir 24 horas con vegetales aromáticos
4. Colar y envasar
5. Congelar en porciones de 500ml

**Precio Sugerido:** Gs. 35.000/litro

**Cliente:** Mismo cliente de Nivel 2 (upsell)

**Por qué esperar:** Necesitás volumen de huesos para que sea rentable. Con 8+ clientes, generás suficientes huesos para hacer batch semanal.

#### BARF para Mascotas {#barf-mascotas}

**Condición de Activación:**
```
EXCEDENTE:recortes (>3kg/semana) + DEMANDA:mascotas (3+ clientes preguntan)
```

**Origen:** Recortes con nervios, grasa excesiva, cartílagos.

**Productos:**
- "Mix Carnívoro" - Hamburguesas crudas balanceadas
- "Hueso Recreativo" - Fémur con tuétano
- Galletas naturales horneadas

**Proceso:**
1. Separar recortes durante procesamiento normal
2. Congelar hasta tener volumen
3. Moler con vísceras baratas (corazón, hígado)
4. Porcionar en 500g

**Precio Sugerido:** Gs. 20.000/kg

**Cliente:** Dueños de perros que buscan alimentación natural.

**Por qué esperar:**
1. Es un cliente DIFERENTE (no tu cliente actual)
2. Requiere investigar regulaciones SENAVE
3. Solo tiene sentido si tenés excedente real, no comprando específicamente

**Señal para activar:** 3 o más clientes te preguntan "¿y los recortes qué hacés con ellos?"

---

#### Grasas Puras {#grasas-puras}

**Condición de Activación:**
```
EXCEDENTE:grasa (>2kg/semana) + CLIENTES:10+
```

**Productos:**
- Sebo Premium (grasa de vaca purificada)
- Schmaltz (grasa de pollo con cebolla)
- Velas Comestibles (grasa + romero)

**Proceso Sebo:**
1. Juntar grasa dura de picanha/bife
2. Derretir lento (2-3 horas)
3. Filtrar 3 veces (filtro de café)
4. Envasar en frascos de vidrio

**Precio Sugerido:** Gs. 25.000/frasco 250ml

**Por qué esperar:** La grasa es un subproducto. Primero necesitás suficiente volumen de clientes generando grasa para que valga la pena el proceso.

---

#### Chicharrón {#chicharron}

**Condición de Activación:**
```
EXCEDENTE:cuero-cerdo (clientes piden cerdo regularmente)
```

**Proceso:**
1. Hervir cuero 30 min
2. Secar al sol o deshidratador (24 hs)
3. Freír rápido en aceite caliente
4. Se infla como nube

**Cliente:** Keto, picadas, snacks

**Por qué esperar:** Requiere que proceses cerdo regularmente. Si no hay demanda de cerdo, no hay cuero.

---

#### Tepache {#tepache}

**Condición de Activación:**
```
EXCEDENTE:cascara-piña (>3 piñas/semana)
```

**Proceso:**
1. Cáscaras + centro de piña
2. Agua + azúcar morena + canela
3. Fermentar 3-5 días
4. Filtrar y embotellar

**Precio Sugerido:** Gs. 15.000/litro

**Por qué esperar:** Necesitás flujo constante de cáscaras. Solo funciona si clientes piden piña regularmente.

---

#### Polvos Umami {#polvos-umami}

**Condición de Activación:**
```
EXCEDENTE:cascaras-vegetales (procesamiento regular de volumen)
```

**Ingredientes:**
- Pieles de cebolla
- Cáscaras de zanahoria
- Piel de tomate
- Recortes de hongos

**Proceso:**
1. Deshidratar en horno bajo (4-6 horas)
2. Moler fino
3. Mezclar con sal marina
4. Envasar en frascos pequeños

**Precio Sugerido:** Gs. 20.000/frasco 100g

**Por qué esperar:** Requiere volumen de procesamiento. Con 5 clientes, no generás suficientes cáscaras.

#### Nivel 3: Comida Cocida {#nivel-3-cocida}

**Condición de Activación:**
```
CLIENTES:15+ Y CAPITAL:10M+ Y PERMISO:cocina-comercial
```

**Qué es:**
Freezer meals - comida terminada lista para calentar.

**Productos:**
- Salsas base (pomodoro, bolognesa)
- Estofados (locro, guiso)
- Proteínas cocidas (pollo desmenuzado, carne mechada)
- Sopas congeladas

**Por qué esperar:**
1. Requiere cocina comercial registrada (INAN)
2. Requiere equipamiento (freezer industrial, ollas grandes)
3. Mayor riesgo de intoxicación = mayor responsabilidad legal
4. Necesitás base de clientes que confíen en vos primero

**Señal para activar:**
- 5+ clientes te preguntan si podés cocinar
- Tenés capital para invertir en equipamiento
- Conseguís espacio de cocina comercial

#### B2B Restaurantes {#b2b-restaurantes}

**Condición de Activación:**
```
TIEMPO:15+ horas libres/semana Y CAPITAL:5M+
```

**Productos:**
- Cebolla picada (bolsa 5kg)
- Papa pelada (bolsa 10kg)
- Ajo pelado (por kg)
- Zanahoria rallada (por kg)

**Por qué esperar:**
1. Márgenes muy bajos (competís con proveedores establecidos)
2. Clientes exigentes (delivery exacto, consistencia perfecta)
3. Requiere vehículo más grande
4. Horarios diferentes (madrugada)

**Señal para activar:**
- Un restaurante te contacta específicamente
- Tenés días de la semana sin trabajo de clientes particulares

```
CLIENTES:20+ Y DEMANDA:talleres (5+ personas preguntan)
```

2. Requiere marketing diferente (Instagram, publicidad)

4. Compite con tu tiempo de servicio a clientes

- Tenés lista de espera de clientes nuevos

#### Red Nutricionistas {#nutricionistas}

**Condición de Activación:**
```
CLIENTES:10+ Y ALIANZA:1+ nutricionista interesado
```

**Modelo:**
- Nutricionista envía dieta del paciente
- Vos cocinás y pesás gramo por gramo
- Cliente paga suscripción mensual
- Nutricionista gana retención de pacientes

**Por qué esperar:**
1. Requiere precisión extrema (gramos exactos)
2. Requiere Nivel 3 (comida cocida)
3. Necesitás nutricionista que confíe en vos

**Señal para activar:**
- Un nutricionista te contacta
- Ya tenés capacidad de Nivel 3

ESTACIÓN:octubre-noviembre Y CAPITAL:3M+

#### Cow-Pooling / Club de la Vaca {#cow-pooling}

**Condición de Activación:**
```
CLIENTES:20+ Y COMUNIDAD:grupo-whatsapp activo
```

**Modelo:**
- 4 familias se juntan para comprar media res
- Vos gestionás compra, faena, reparto
- Cobrás tarifa de servicio

**Por qué esperar:**
1. Logística de pesadilla (coordinar 4 familias)
2. Bajo margen por persona
3. Requiere comunidad establecida que confíe

**Señal para activar:**
- Tenés grupo de WhatsApp con 50+ personas activas
- 4+ personas te preguntan por compras grupales
